Created an Android application that allows user to create and manage trips among friends.
Provided features like user profile, send/accept friend requests, create trips, chat rooms with people interested in the trip by using Google Firebase and using its features like Authentication, Realtime Database and Storage. Used Google Maps API which made users to search, add and delete places for every trip and show the round-trip details on the map using Google Directions API. 
Technology used: Android Studio 2.0, Android SDK 7.0, Google Firebase
